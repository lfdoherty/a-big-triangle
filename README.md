fast-big-triangle
==============

See [https://github.com/mikolalysenko/a-big-triangle].

## Install

```sh
npm install fast-big-triangle
```

## API

### `require("fast-big-triangle")(gl)`
Draws a fullscreen triangle.

* `gl` is a WebGL context
